# TripRadar Travel Platform

A modern travel planning and budget tracking web application that helps users create and manage travel itineraries, monitor expenses, and discover destinations.

## Core Features

- Trip planning and itinerary management
- Budget tracking and expense monitoring
- Travel search and destination discovery
- User authentication with OAuth support (Google, GitHub, Microsoft)
- Responsive design with dark mode support
- User profile management

## Target Users

Travelers who want to organize their trips, track budgets, and plan itineraries in a modern, user-friendly interface.

## Backend API

- Development API: `https://api-dev.tripradar.io`
- Database: PostgreSQL (TripRadarServerDb)
- API documentation available via Swagger
